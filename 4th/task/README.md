## 課題6

引数で受け取った配列の値を2倍にして返す高階関数を作成してください。
出力結果は `console.log` を使用してください。
高階関数の返り値は下記のような結果を期待します。

```javascript
const num = [1,2,3,4,5];      // この配列を引数に渡すと
const result = [2,4,6,8,10];  // それぞれの数値が2倍になった配列が返ってくる
```
